---

## Proyectos destacados

### 🟣 ValoShop — Tienda de VALORANT + Account Checker
Web para **ver la tienda** que te ha tocado en tu cuenta de VALORANT, además de un **checker** para compartir que skins tienes.  
**Stack:** JavaScript, TypeScript, HTML, CSS  
🔗 https://valoshop.kirakohler.es

### 🧩 K1R4FN — Hybrid de Fortnite
Es un servidor **híbrido** de Fortnite, el cuál te permite tener **cualquier cosmético** del juego incluso aunque este no haya salido.  
(Parte del ecosistema incluye una API interna para el proyecto.)

### 👾 ClashRPC — Discord Rich Presence para Clash Royale
Programa para mostrar un **Rich Presence** en Discord vinculado a tu cuenta de **Clash Royale**.  
**Stack:** Rust + JavaScript
